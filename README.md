# E-commerce Sales Performance Dashboard (MIS Report)

## 📌 Project Overview

This project is an end-to-end MIS reporting solution built in Microsoft Excel using raw e-commerce sales data. The objective was to clean and structure unorganized data, develop a reporting system, and create an interactive dashboard to track key business metrics.

The project simulates a real-world MIS Executive role where the focus is on maintaining data accuracy, generating reports, and providing actionable insights to support business decisions.

---

## 🎯 Business Objective

* Track monthly and category-wise sales performance
* Compare actual performance against targets
* Identify top-performing states, products, and customers
* Highlight underperforming areas for business improvement

---

## 📊 Key Features

* Data cleaning and preparation using Excel functions
* Data merging using VLOOKUP and structured tables
* KPI tracking (Revenue, Profit, Orders, Average Order Value)
* Pivot table-based reporting for performance analysis
* Interactive dashboard with slicers (Month & Category)
* Target vs Actual analysis with variance and achievement %
* Conditional formatting for quick performance insights
* VBA automation for one-click dashboard refresh

---

## 📈 Key Insights

* January recorded the highest revenue (~₹61K), indicating a strong sales period
* Electronics category generated the highest overall revenue (~₹1.65L+)
* A small group of top customers contributed a significant portion of total sales
* Certain categories underperformed compared to targets, highlighting scope for improvement
* Sales performance varied significantly across states, with a few states driving most of the revenue
